Report skipped tests in Jest

## Installation

Using yarn:

```bash
$ yarn add -D jest-skipped-reporter
```

Using npm:

```bash
$ npm i -D jest-skipped-reporter
```

## Usage

Jest CLI:

```bash
jest --reporters jest-skipped-reporter
```

Jest Config:

```json
{
  "reporters": ["jest-skipped-reporter"]
}
```
